# Music-Client

This is an Angular UI to [Music-JS](https://gitlab.com/snobili/music-js) or [Music-Laravel](https://gitlab.com/snobili/music-laravel).

## Design rules

- Lazy-loading of Angular modules
- Reactive state with [NgRx](https://ngrx.io/)
- Pagination of API calls and data display

## Running

```
npm install
npm start
```

## Building

```
npm run build
```
